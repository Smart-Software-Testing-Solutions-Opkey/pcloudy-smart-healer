# Contributing to SmartHealer

Thank you for your interest in contributing to SmartHealer! We appreciate your support and enthusiasm for this project.

## Current Status

SmartHealer is currently in its initial development phase. While we are not accepting code contributions at this time, we value your input and engagement with the project.

### What We Accept Now

- **Bug Reports**: Found a bug? Please report it via [GitHub Issues](../../issues)
- **Feature Requests**: Have ideas for improvements? Share them via [GitHub Issues](../../issues)
- **Documentation Feedback**: Spotted errors or unclear documentation? Let us know via [GitHub Issues](../../issues)

### What We're Not Accepting Yet

- Pull requests with code changes
- Direct code contributions

We plan to open the project for full community contributions soon. The timeline for this is currently **TBD**.

## Stay Updated

We encourage you to:

- **Star** this repository to show your support
- **Watch** this repository to get notified when we open contributions
- Check back regularly for updates on our contribution timeline

## How to Report Bugs

When reporting bugs, please include:

1. **Description**: Clear description of the issue
2. **Steps to Reproduce**: Detailed steps to reproduce the behavior
3. **Expected Behavior**: What you expected to happen
4. **Actual Behavior**: What actually happened
5. **Environment**:
   - OS and version
   - Node.js version
   - Appium version (if applicable)
   - SmartHealer version
6. **Screenshots/Logs**: If applicable, add screenshots or relevant log outputs

## How to Request Features

When requesting features, please include:

1. **Problem Statement**: What problem does this feature solve?
2. **Proposed Solution**: Your suggested approach
3. **Alternatives Considered**: Other solutions you've thought about
4. **Additional Context**: Any other relevant information

## Code of Conduct

This project adheres to the [Contributor Covenant Code of Conduct](https://www.contributor-covenant.org/version/2/1/code_of_conduct/). By participating, you are expected to uphold this code. Please report unacceptable behavior via GitHub Issues or contact the project maintainers.
